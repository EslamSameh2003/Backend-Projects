# Personal Blogging Platform API

This is a backend REST API for a Personal Blogging Platform built with Node.js, Express, and Sequelize.  
It provides user authentication, blog management, and basic search functionality.

---

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Create / Edit / Delete Blogs
- View All Blogs
- Sequelize ORM with SQL database

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Sequelize
- MySQL 
- JWT (JSON Web Token)

---
## ⚙️ Installation

```bash
git clone https://github.com/EslamSameh2003/Backend-Projects/tree/personal-blogging-platform

```
